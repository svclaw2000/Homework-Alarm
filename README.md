# 과제 알리미

## 📄 Introduction
과제가 있는 것을 잊고 마감 직전에 급하게 진행하는 경우를 방지하기 위한 알람 어플

## 🔑 Features
- 등록해 놓은 과제 마감시간까지 상단바 알림
- 설정에 따른 마감 전 알림
- 마감시간이 지나면 자동 소멸
